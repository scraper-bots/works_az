Təsvir

PAŞA Property Management “Texniki dəstək üzrə mütəxəssis” vakansiyası üzrə işə qəbul elan edir.

 

Ölkənin aparıcı daşınmaz əmlak idarəetmə şirkətlərindən biri olan PAŞA Property Management, Sizə karyera yolunda uğur qazanmaq və peşəkar komandada özünüzü inkişaf etdirmək imkanı təqdim edir. Biz, öyrənməyə meyilli, pozitiv düşüncəli və işinə yaradıcı yanaşan, motivasiyalı əməkdaşlar axtarırıq.

Tələblər

Vəzifə öhdəlikləri:

  • Kompüterlərin və təşkilatı texnikanın daim işlək vəziyyətdə saxlanmasını təmin edir;
  • Lokal kompüter şəbəkəsinin fasiləsiz işləməsi və saz vəziyyətdə saxlanmasını təmin edir, şəbəkə və informasiya təhlükəsizliyinin inkişafı üçün ardıcıl tədbirlər görür;
  • Videomüşahidə (CCTV) quraşdırılmasına, sazlanmasına və onlara texniki xidmət işlərinə nəzarət edir;
  • Şirkət üzrə informasiya sistemlərinin qurulmasını, onların etibarlı və davamlı fəaliyyətini, təhlükəsizliyini və işlək vəziyyətdə saxlanmasını, dövri yenilənməsini təmin edir;
  • Şirkətin kompüter və server avadanlıqlarında istifadə olunan ümumi təyinatlı proqram təminatının işlək vəziyyətdə saxlanmasını və onun dövri yenilənməsini təmin edir;
  • İnformasiya sistemlərinin istismarı ilə əlaqədar mütəmadi olaraq monitorinqlər və təhlillər aparır, həmin məsələlərin həlli üzrə təkliflər hazırlayır, aşkar edilən problem və nasazlıqları aradan qaldırır;
  • Şirkətin informasiya sistemlərindən fasiləsiz istifadənin təmin edilməsi məqsədi ilə dəstək xidmətini təşkil edir və s.


Vakansiya tələbləri:

  • Ali təhsil arzu olunandı
  • Sahə üzrə minimum 2 il iş təcrübəsi;
  • Azərbaycan dilində səlis danışıq və yazı bacarığı (rus və ingilis dili biliyi üstünlükdür);
  • Özünü inkişafa meyillilik, ünsiyyət, komandada işləmə, problem həlletmə bacarıqları.
